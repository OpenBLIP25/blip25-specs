## Notation Key

Each burst = 180 dibit symbols. Each symbol carries Bit 1 (MSB) and Bit 0 (LSB).

| Notation | Meaning |
|----------|--------|
| `c0(n)`..`c3(n)` | Vocoder codeword bits (c0 = most protected, c3 = least) |
| `Z` | Ramp/guard zero symbol (single dibit) |
| `P1(n)` / `P2(n)` | Pilot sequence symbols (single dibit) |
| `S(n)` | Sync sequence symbols (single dibit) |
| `DUID(n)` | Data Unit ID bits |
| `DUIDparity(n)` | DUID parity bits |
| `ESS-B(n)` | Encryption Sync Signal type B bits |
| `RS_parity(n)` | Reed-Solomon FEC parity bits |
| `ISCH(n)` | Inter-Slot Signaling Channel bits |
| `IEMI(n)` | Inbound EMI signaling payload |
| `S-OEMI(n)` | Outbound EMI with sync signaling payload |
| `I-OEMI(n)` | Outbound EMI without sync signaling payload |

Single-dibit fields (Z, P1, P2, S) show the same value in both Bit 1 and Bit 0 columns — the value represents the dibit symbol, not individual bits.
